===========
L10n FR UBL
===========

This is a small glue module between the modules *base_ubl* and *l10n_fr_siret*: it adds the SIRET number in the node *Party / PartyIdentification / ID* of the UBL XML files generated by Odoo. This is required if you use `Chorus Pro <https://www.chorus-pro.gouv.fr>`_ to invoice the French State.

Configuration
=============

No configuration needed.
