==========================
Product Variant CSV Import
==========================

This module adds a menu entry in

Sales > Configuration > Product Categories and attributes > Product Template CSV Import

that will work with the import of CSV file product.template.csv that contains variants.

Usage
=====

See 'sample_csv' folder for examples: in product.template.csv, for every variant (product.product), you can use 'attribute_value_ids' field writing its values in the same way as any other many2many field
